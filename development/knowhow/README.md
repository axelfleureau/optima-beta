# Righello Development Know-How

Questa cartella raccoglie micro-skill operative nate da problemi reali incontrati nello sviluppo di Optima e dei prodotti Righello.

Obiettivo: ogni volta che impariamo una regola ricorrente, trasformarla in una checklist riusabile prima di progettare, implementare, testare o deployare.

## Come usarla

- Prima di iniziare una nuova feature, cerca la skill pertinente.
- Durante review e audit, usa le checklist come criteri minimi di qualita.
- Quando emerge un nuovo pattern importante, aggiungi un file Markdown dedicato.
- Mantieni ogni file breve, operativo e aggiornabile.
