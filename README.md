# Optima — AI-Powered Digital Marketing Automation Platform

> Built by [Righello S.r.l.](https://www.wearerighello.com) — Innovative Startup, Italy 🇮🇹

## What is Optima?

Optima is a multi-tenant SaaS platform that integrates AI into real operational workflows
for SMEs: campaign performance analysis in real-time, automated content generation,
budget optimization, and no-code/low-code automation pipelines.

The platform is in active development as the core business of Righello S.r.l.,
a registered Italian innovative startup (CCIAA Pordenone-Udine, REA PN-376408).

## Tech Stack

- **Frontend**: React, TypeScript
- **Backend**: Node.js / Python services
- **AI layer**: Large language models for content generation and decision support
- **Automation**: No-code/low-code pipeline orchestration
- **Infrastructure**: Cloud-native, multi-tenant architecture
